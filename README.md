apify
============

API documentation generator

Installation
-----------
`[sudo] npm install -g apify`

Usage
-----------
`apify --source <source-tree-root> --output <output-folder> [--title <title>] [--config <source-to-config-file]`
`apify --config <source-to-config-file>`
